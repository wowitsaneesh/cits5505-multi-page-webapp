Introduction

This tutorial explains the three core technologies used in front-end web development: HTML, CSS, and JavaScript. These technologies work together to create webpages that are structured properly, styled clearly, and made interactive for users.

HTML provides the structure of a webpage. CSS controls how that structure looks. JavaScript adds behaviour so that the page can respond to the user. A modern webpage usually uses all three at the same time.

For example, a quiz page may use HTML to create the form and questions, CSS to improve the design and readability, and JavaScript to load questions, check answers, calculate the score, and update the result without reloading the page.

HTML Fundamentals

HTML stands for Hyper Text Markup Language. It is used to structure the content of a webpage. The browser reads HTML tags and uses them to understand what each part of the page represents. HTML is mainly about meaning and structure, not visual styling.

For example, headings tell the browser that a piece of text is a title or subheading. Paragraph tags mark normal blocks of text. Link tags create clickable navigation. Image tags display pictures. These elements help organise the page clearly for both users and browsers.

Common HTML Tags

  • <h1> to <h6> are used for headings
  • <p> is used for paragraphs
  • <a> is used for hyperlinks
  • <img> is used for images
  • <ul> and <ol> are used for lists

HTML Example 1

<h1>Welcome</h1>
<p>This is a paragraph on the page.</p>

In this example, the heading provides a main title, while the paragraph gives supporting information underneath it. This is a simple example of page structure.

HTML Example 2

<a href="quiz.html">Go to the quiz</a>
<img src="images/dawg.jpg" alt="Example image">

The first line creates a clickable link to another page. The second line inserts an image into the webpage. The alt attribute gives alternative text, which is important for accessibility and for cases where the image cannot load.

CSS Fundamentals

CSS stands for Cascading Style Sheets. It is used to control the appearance of a webpage. While HTML provides the structure, CSS changes the visual presentation of that structure. CSS can change colours, spacing, fonts, borders, and layout.

CSS works by selecting HTML elements and applying style rules to them. A selector can target an element by tag name, by class, or by id. For example, p targets all paragraph elements, .container targets elements with the class container, and #main-header targets the element whose id is main-header.

What CSS Can Control

  1. Text colour and background colour
  2. Spacing, including padding and margins
  3. Font size, style, and emphasis
  4. Border styling and box appearance
  5. Layout and alignment of content

CSS Example 1

p {
    color: blue;
    font-size: 18px;
}

This CSS rule changes the text colour of every paragraph to blue and makes the text larger. This shows how one CSS rule can affect all matching elements.

CSS Example 2

#main-header {
    background-color: blue;
}

.container {
    padding: 20px;
}

The selector #main-header is an id selector. It targets one specific element with that id. The selector .container is a class selector. In Bootstrap, .container is commonly used to keep page content centred and spaced neatly.

JavaScript Fundamentals

JavaScript is used to add behaviour and interactivity to webpages. Without JavaScript, a page is mostly static. With JavaScript, the page can react when a user clicks a button, enters text, submits a form, or interacts with the page in some other way.

JavaScript can also change the content already on the page. It can show alerts, ask the user for input, calculate results, store data in the browser, and load information from files or APIs. In this project, JavaScript is what makes the quiz interactive.

Common JavaScript Uses

  • Showing popup messages with alert()
  • Asking for input with prompt()
  • Printing information with console.log()
  • Saving data using localStorage
  • Updating page content dynamically

JavaScript Example 1

alert("Welcome to JavaScript!");

This line shows a popup message to the user. It is one of the simplest examples of JavaScript interactivity.

JavaScript Example 2

let userName = prompt("What is your name?");
localStorage.setItem("name", userName);

The function prompt() asks the user to type in some input. The feature localStorage can then save small pieces of information inside the browser, such as quiz attempts or user preferences.
