Introduction

This tutorial introduces the three core technologies used in front-end web development: HTML, CSS, and JavaScript.

HTML provides the structure of a web page, CSS controls the appearance, and JavaScript adds behaviour and interactivity.

HTML Fundamentals

HTML is used to structure content on a web page.

Example HTML tags

  • <h1> to <h6> for headings
  • <p> for paragraphs
  • <a> for links
  • <img> for images

Code Example

<h1>Hello World</h1>
<p>This is a paragraph.</p>

CSS Fundamentals

CSS is used to control colours, spacing, fonts, layout, and visual style.

  1. CSS can change text colour
  2. CSS can set page layout
  3. CSS can improve readability and design

Code Example

p {
    color: blue;
}

JavaScript Fundamentals

JavaScript is used to make web pages interactive.

It can respond to user actions, update content, and process data.

Code Example

alert("Welcome to JavaScript!");
