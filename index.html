Introduction

This tutorial introduces the three core technologies used in front-end web development: HTML, CSS, and JavaScript. These three technologies work together to create webpages that are structured correctly, styled clearly, and made interactive for users.

HTML provides the structure of a webpage. CSS controls how the page looks. JavaScript adds behaviour and interactivity. By the end of this page, you should understand the basic ideas needed to answer the quiz questions on the next page.

In simple terms, HTML tells the browser what the content is, CSS tells the browser how the content should look, and JavaScript tells the browser how the page should respond when the user clicks, types, or interacts with something.

HTML Fundamentals

HTML stands for Hyper Text Markup Language. It is used to build the structure of a webpage. HTML uses tags to mark different types of content, such as headings, paragraphs, images, and links.

Important HTML Tags

  • <h1> to <h6> are used for headings
  • <p> is used to create a paragraph
  • <img> is used to insert an image into a webpage
  • <a> is used to create a hyperlink

Example HTML Code

<h1>Welcome</h1>
<p>This is a paragraph on the page.</p>
<a href="quiz.html">Go to the quiz</a>
<img src="images/dawg.jpg" alt="Example image">

The code above shows four useful HTML elements. The heading gives the page a title, the paragraph displays a block of text, the hyperlink lets the user move to another page, and the image tag displays a picture. These are some of the most common building blocks of a webpage.

CSS Fundamentals

CSS stands for Cascading Style Sheets. It is used to control the appearance of a webpage. While HTML provides structure, CSS changes the visual presentation, including colours, spacing, fonts, borders, and layout.

What CSS Does

  1. Changes colours and backgrounds
  2. Controls spacing such as margin and padding
  3. Changes font size, font weight, and text style
  4. Improves layout and visual organisation
  5. Helps pages look cleaner and easier to read

Useful CSS Selectors

  • p selects all paragraph elements
  • #main-header selects the element with the id main-header
  • .container selects elements with the class container

Example CSS Code

#main-header {
    background-color: blue;
}

p {
    color: black;
}

.container {
    padding: 20px;
}

In the example above, #main-header is an id selector. It targets one specific element whose id is main-header. The selector .container is a class selector. In Bootstrap, .container is commonly used to keep page content centred and spaced neatly across different screen sizes.

JavaScript Fundamentals

JavaScript is used to add behaviour and interactivity to webpages. Without JavaScript, a page is usually mostly static. With JavaScript, the page can respond when users click buttons, enter information, submit forms, or trigger browser events.

What JavaScript Is Used For

  • Showing alerts and popup messages
  • Asking the user for input with prompt()
  • Changing page content dynamically
  • Checking quiz answers and calculating scores
  • Loading data and saving information in the browser

Example JavaScript Code

alert("Welcome to JavaScript!");
let userName = prompt("What is your name?");
localStorage.setItem("name", userName);

The function alert() shows a popup message. The function prompt() asks the user to type in some input. The browser feature localStorage can be used to save small pieces of information on the client side, such as quiz attempts, names, or settings.

In this project, JavaScript is what makes the quiz interactive. It checks answers, shows the score, displays whether the user passed, stores previous attempts, and loads question data dynamically from a file.
