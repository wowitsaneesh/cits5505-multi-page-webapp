Introduction

This tutorial introduces the three core technologies used in front-end web development: HTML, CSS, and JavaScript. Together, these technologies allow developers to create websites that are structured properly, visually clear, and interactive for users.

HTML provides the structure and meaning of the content on a webpage. CSS controls how that content looks, including layout, colour, spacing, and fonts. JavaScript adds behaviour and interactivity, allowing pages to respond to user actions and update dynamically in the browser.

A successful webpage normally requires all three technologies to work together. For example, a quiz page may use HTML to define the form, CSS to improve the presentation, and JavaScript to check answers, calculate scores, and update the displayed result.

HTML Fundamentals

HTML stands for Hyper Text Markup Language. It is used to structure content on a webpage so the browser knows what each part of the page represents. HTML is not mainly responsible for appearance or behaviour. Instead, it provides the backbone of the page.

Common HTML Tags

  • <h1> to <h6> for headings
  • <p> for paragraphs
  • <a> for hyperlinks
  • <img> for images
  • <ul> and <ol> for lists
  • <form> and <input> for forms

Example HTML Code

<h1>Welcome</h1>
<p>This is a paragraph on the page.</p>
<a href="quiz.html">Go to the quiz</a>

In the example above, the heading gives the page a clear title, the paragraph presents information, and the hyperlink allows the user to move to another page. This shows how HTML connects structure and navigation.

CSS Fundamentals

CSS stands for Cascading Style Sheets. It is used to control the appearance of a webpage. While HTML gives the page structure, CSS makes the page visually appealing and easier to read.

What CSS Can Control

  1. Text colour and background colour
  2. Spacing, margins, and padding
  3. Font size and style
  4. Page layout and positioning
  5. Responsive presentation across different screen sizes

Example CSS Code

p {
    color: blue;
    font-size: 18px;
}

This CSS rule applies styling to all paragraph elements. It changes the text colour to blue and increases the font size. CSS works by selecting HTML elements and applying style rules to them.

JavaScript Fundamentals

JavaScript is used to add behaviour and interactivity to webpages. It allows the page to respond when a user clicks a button, enters text, submits a form, or loads the page. It can also change the content of the page without needing to reload it fully.

Common Uses of JavaScript

  • Showing messages and alerts
  • Reading user input
  • Changing text on the page dynamically
  • Checking quiz answers
  • Loading data from a file or API

Example JavaScript Code

alert("Welcome to JavaScript!");

This example shows a simple JavaScript instruction that displays a popup message to the user. JavaScript can be much more powerful than this, but even basic commands help demonstrate how a webpage can become interactive instead of remaining static.
