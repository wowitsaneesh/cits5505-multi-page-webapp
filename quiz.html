Current Progress and Future Features

This quiz page can currently do the following:

  1. Load 10 quiz questions from a local JSON file using AJAX
  2. Display the loaded questions and answer choices on the page
  3. Randomise question order each time the quiz loads
  4. Check all selected answers using JavaScript
  5. Display score, percentage, and pass/fail on the page
  6. Save quiz attempts in local storage
  7. Show previous attempts when the page loads
  8. Warn the user if they try to leave after starting but before submitting

Later work will allow us to add:

  1. Reward content from a public API
  2. Even richer quiz feedback and presentation
