Current Progress and Future Features

This quiz page can currently do the following:

  1. Load one quiz question from a local JSON file using AJAX
  2. Display the loaded question and answer choices on the page
  3. Check the selected answer using JavaScript
  4. Display the result on the page
  5. Save the latest quiz attempt in local storage
  6. Show the latest saved attempt when the page loads

Later lectures will allow us to add:

  1. Multiple dynamically rendered quiz questions
  2. Randomised question order
  3. Automatic percentage and pass/fail calculation
  4. Reward content from a public API
  5. Full quiz attempt history display
  6. Navigation warning if the user leaves mid-quiz
